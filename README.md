🎓 Campus Management System (CMS)
📌 Project Title
Campus Management System

📖 Project Description
Placement Management System is a web-based application developed to automate and manage the college placement process.
It connects students and placement cell through a centralized digital platform.

The system allows students to register, login, view job listings, apply for jobs, and track application status.

🎯 Objectives
To automate the placement process

To reduce manual paperwork

To provide transparency in job application tracking

To maintain centralized student and job data

Admin (Placement Cell)
